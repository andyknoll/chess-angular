About this app


Presentation for Front End Developer Class - Jan 10, 2019

Eric Rinella - Instructor and Benevolent Leader


This app is about Angular Routing. It is written as a true Angular app which uses routing to demonstrate routing :)

All navigation in this small app is accomplished using routing and the browser history.

While researching the topic of Angular routing, I explored several third-party tutorials and the Angular documentation itself. I used this project not only to learn more about routing but to also experience all that goes into assembling an actual Angular app (angular-cli, components, modules, etc.) and the logistics involved.

I decided to use what I was learning to put together an Angular app for the experience and the gotchas. Due to time constraints I was not able to demonstrate all that I wanted but I will most likely add to this presentation at a later time and polish it up a bit more.
